# Retail Sales Forecasting & Time Series Analysis
### Using Python,ARIMA,Seasonal Decomposition & Power BI Visualization
## Project Overview

This project analyzes historical retail sales data to uncover sales trends, seasonal patterns, and future demand using Time Series Analysis and Forecasting techniques.

Python was used for data cleaning, exploratory data analysis (EDA), seasonal decomposition, and ARIMA forecasting, while Power BI was used to create interactive dashboards for business insights and decision-making.

## Business Objectives

- Analyze historical sales performance
- Identify long-term sales trends
- Detect seasonal demand patterns
- Forecast future sales using ARIMA
- Support inventory and business planning decisions
